Getting Started

Welcome

A stunning visual experience for minimal loving creatives.

This template is a responsive and retina-ready HTML5 website with bootstrap grid system layout. It is optimized for mobile touch and swipe.

Features

  • Valid HTML5 / CSS3 pages
  • Perfectly responsive
  • Retina-Ready
  • Cross browser compatible – tested in IE 9,10,11; Firefox; Safari; Chrome; Edge and Opera.
  • Well formatted and commented, so it should be easy to adapt and extend
  • Working ajax contact form with validation and working PHP mailer script
  • Hundreds of Retina-Ready Icons included http://fontawesome.io

How to add columns?

Here you can utilize the grid system of bootstrap, which is fully integrated.
For more information on how this system works, please take a look at http://getbootstrap.com/css/#grid

Need more icons?

You find the codes in styles.css under section 10: Icons. If you need more font-awesome icons, then you’ll find what you need on this cheat sheet.

How to change the email address to which emails are sent from the contact form?

Find in contact.php: $address = 'support@yoursite.com';
There you can enter your email address. A few lines further down, you can also change the text which is displayed in the email.

How to change the google maps?

Go to custom.js find line no:85 and change longitude and latitude according to your map

center:new google.maps.LatLng(40.712775,-74.005973)
